# Инструкция по работе с Git

Git - это система управления версиями с распределенной архитектурой.

## Создание нового репозитория

Чтобы создать новый репозиторий (инициализировать) нужно ввести комманду:

    git init

## Проверка актуального состояния репозитория

Для того, чтобы проверить в каком сотоянии находится репозиторий нужно ввести комманду:

    git status

## Сохранение снимка текущего состояния проекта в истории коммитов

Для того, чтобы добавить все файлы из указанного каталога в индекс нужно ввести комманду:

    git add
